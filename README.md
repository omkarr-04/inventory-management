 KK Auto — Workshop & Inventory Management System

A professional Android application built for commercial vehicle repair workshops.

The application combines:
- Inventory Management
- Job Card Workflow
- Billing & Invoice Generation
- Payment Tracking
- Real-Time Multi-User Sync
- Barcode / QR Scanning
- Dashboard Analytics

Built using:
- Kotlin
- Jetpack Compose
- Firebase Firestore
- MVVM Architecture

---

 Features

 Inventory Management

- Real-time stock synchronization
- Low stock alerts
- Search & filtering
- Barcode / QR support
- Adjustable stock thresholds
- INR currency support

---

 Job Card System

- Create repair jobs
- Add multiple parts
- Auto stock deduction
- Vehicle workflow tracking
- Status updates:
    - Pending
    - In Progress
    - Completed
    - Delivered

---

 Billing & Payments

- GST-ready invoice system
- Labor charge support
- Payment tracking:
    - Paid
    - Partial
    - Unpaid
- Due date reminders
- PDF invoice generation
- WhatsApp invoice sharing

---

 Multi-User Real-Time Sync

All devices stay synchronized automatically using Firebase Firestore.

If one user updates:
- stock
- jobs
- payments
- invoices

all other connected devices instantly receive updates.
